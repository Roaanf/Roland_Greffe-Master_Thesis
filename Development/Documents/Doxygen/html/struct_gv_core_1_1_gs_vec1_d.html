Detailed Description

template<uint Tr>
struct GvCore::GsVec1D< Tr >

The GsVec1D struct provides the concept of a uniform 3D resolution.

This is a specialization of a GsVec3D 3D resolution where each dimension is equal. This is notably used to define space resolution/extent of node tiles and bricks of voxels.
