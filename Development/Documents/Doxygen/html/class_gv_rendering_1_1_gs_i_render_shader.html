Detailed Description

template<typename TDerived>
class GvRendering::GsIRenderShader< TDerived >

The GsIRenderShader class provides the main interface for shading the data structure.

This class is the base class for all shaders.

Parameters
TDerivedthe class used to implement the shader functions.
