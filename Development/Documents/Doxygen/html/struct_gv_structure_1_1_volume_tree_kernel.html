struct GvStructure::VolumeTreeKernel< DataTList, NodeTileRes, BrickRes, BorderSize >

The VolumeTreeKernel struct provides the interface to a GigaVoxels data structure on device (GPU).

This is the device-side associated object to a GigaVoxels data structure.
