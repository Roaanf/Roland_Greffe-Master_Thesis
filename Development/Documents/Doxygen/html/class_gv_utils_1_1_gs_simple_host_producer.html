Detailed Description

template<typename TKernelProducerType, typename TDataStructureType, typename TDataProductionManager>
class GvUtils::GsSimpleHostProducer< TKernelProducerType, TDataStructureType, TDataProductionManager >

The GsSimpleHostProducer class provides the mecanisms to produce data following data requests emitted during N-tree traversal (rendering).

It is the main user entry point to produce data from CPU, for instance, loading data from disk or procedurally generating data.

This class is implements the mandatory functions of the GsIProvider base class.
