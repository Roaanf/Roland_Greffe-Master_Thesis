Retrieve the brick file name. An example of GigaVoxels brick file could be : "fux_BR8_B1_L0_C0_uchar4.bricks" where "fux" is the name, "8" is the brick width BR, "1" is the brick border size B, "0" is the level of resolution L, "0" is the data channel index C, "uchar4" the data type name and "bricks" is the file extension.

Parameters
pNamename of the data file
pLeveldata structure level of resolution
pBrickWidthwidth of bricks
pDataChannelIndexdata channel index
pDataTypeNamedata type name
Returns
the brick file name in GigaVoxels format.
