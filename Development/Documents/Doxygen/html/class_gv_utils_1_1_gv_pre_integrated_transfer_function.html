Detailed Description

The GvPreIntegratedTransferFunction class provides an implementation of a pre-integrated transfer function on the device.

Transfer function is a mathematical tool used tu map an input to an output. In computer graphics, a volume renderer can use it to map a sampled density value to an RGBA value.
