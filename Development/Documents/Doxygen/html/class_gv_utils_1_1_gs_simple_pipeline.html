Detailed Description

template<typename TShaderType, typename TDataStructureType, typename TCacheType>
class GvUtils::GsSimplePipeline< TShaderType, TDataStructureType, TCacheType >

The GsSimplePipeline class provides the mecanisms to produce data following data requests emitted during N-tree traversal (rendering).

It is the main user entry point to produce data from CPU, for instance, loading data from disk or procedurally generating data.

This class is implements the mandatory functions of the GsIProvider base class.
