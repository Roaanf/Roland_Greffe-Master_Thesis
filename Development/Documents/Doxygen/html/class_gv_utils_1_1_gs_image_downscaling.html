Detailed Description

The GsImageDownscaling class provides interface to downscale image during rendering.

To improve performance, this class reduce the number of fragments by rendering the image into an offscreen render target with lower resolution (for example, with half the resolution x and y) and zoom up the image when displaying it on the screen.
