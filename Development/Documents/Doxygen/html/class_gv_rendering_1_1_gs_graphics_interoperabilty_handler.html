Detailed Description

The GsGraphicsInteroperabiltyHandler class provides interface to.

Some resources from OpenGL may be mapped into the address space of CUDA, either to enable CUDA to read data written by OpenGL, or to enable CUDA to write data for consumption by OpenGL.
