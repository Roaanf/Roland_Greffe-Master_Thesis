Detailed Description

template<typename T>
class GvCore::GsDeviceTexturingMemory< T >

The GsDeviceTexturingMemory class provides features to manipulate device memory array.

3D Array manipulation class located in device texture memory. It is not the same as linear memory array. Textures and surfaces should be bound to array in order to read/write data.
