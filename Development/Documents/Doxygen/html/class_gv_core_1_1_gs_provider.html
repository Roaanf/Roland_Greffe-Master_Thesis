Detailed Description

template<typename TDataStructure, typename TDataProductionManager>
class GvCore::GsProvider< TDataStructure, TDataProductionManager >

The GsProvider class provides the mecanisms to produce data following data requests emitted during N-tree traversal (rendering).

This class is the base class for all host producers.

It is the main user entry point to produce data from CPU, for instance, loading data from disk or procedurally generating data.
