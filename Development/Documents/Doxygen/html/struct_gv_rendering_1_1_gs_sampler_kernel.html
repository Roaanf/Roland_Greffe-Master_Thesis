Detailed Description

template<typename VolumeTreeKernelType>
struct GvRendering::GsSamplerKernel< VolumeTreeKernelType >

The GsSamplerKernel struct provides features to sample data in a data stucture.

The rendering stage is done brick by brick along a ray. The sampler is used to store useful current parameters needed to fecth data from data pool.

Parameters
VolumeTreeKernelTypethe data structure to sample data into.
