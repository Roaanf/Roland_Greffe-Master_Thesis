Detailed Description

template<typename NodeTileRes, typename LocCodeArrayType, typename LocDepthArrayType>
struct GvCore::PageTable< NodeTileRes, LocCodeArrayType, LocDepthArrayType >

The PageTable struct provides localisation information of all elements of the data structure.

It is used to retrieve localization info (code + depth), .i.e 3D world position of associated node's region in space, from nodes stored in the Cache Management System.

Parameters
NodeTileResNode tile resolution
LocCodeArrayTypeType of array storing localization code (ex : GsLinearMemory< LocalizationInfo::CodeType >)
LocDepthArrayTypeType of array storing localization depth (ex : GsLinearMemory< LocalizationInfo::DepthType >)
