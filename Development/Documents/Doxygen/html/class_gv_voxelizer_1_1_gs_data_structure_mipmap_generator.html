Detailed Description

The GsDataStructureMipmapGenerator class provides an implementation of a scene voxelizer with ASSIMP, the Open Asset Import Library.

It is used to load a 3D scene in memory and traverse it to retrieve useful data needed during voxelization (i.e. vertices, faces, normals, materials, textures, etc...)
