Detailed Description

template<uint Trx, uint Try, uint Trz>
struct GvCore::GsVec3D< Trx, Try, Trz >

The GsVec3D struct provides the concept of a 3D resolution.

This is notably used to define space resolution/extent of node tiles and bricks of voxels.

Note
All members are computed at compile-time.
