Detailed Description

template<class NodeTileRes, class BrickFullRes, class NodeAddressType, class BrickAddressType, class TPriorityPoliciesManager>
struct GvStructure::GsDataProductionManagerKernel< NodeTileRes, BrickFullRes, NodeAddressType, BrickAddressType, TPriorityPoliciesManager >

The GsDataProductionManagerKernel struct provides methods to update buffer of requests on device.

Device-side object used to update the buffer of requests emitted by the renderer during the data structure traversal. Requests can be either "node subdivision" or "load brick of voxels".
